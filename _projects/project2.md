This is a course project of Embedded System, about embedded system development of handwritten digit recognition based on STM32H747I-DISCO. A basic CNN structure is used to detect and classify digits, which performs very well on the MNIST Dataset. We also use CubeAI as a bridge between neural networks and embedded systems.

 ![demo](/images/project2-demo.png)

 More details, like code, report and gif demo, can be found here [Link](https://github.com/blackspiderrr/Embedded_Development_of_Handwritten_Digit_Recognition).
